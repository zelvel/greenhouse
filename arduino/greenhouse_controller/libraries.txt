Required Arduino Libraries:

1. DHT sensor library by Adafruit
   - Install from Library Manager: "DHT sensor library"
   - Version: 1.4.4 or later

2. Adafruit Unified Sensor
   - Install from Library Manager: "Adafruit Unified Sensor"
   - Version: 1.1.9 or later
   - Required by DHT library

3. BH1750
   - Install from Library Manager: "BH1750"
   - Version: 1.3.0 or later

4. ArduinoJson
   - Install from Library Manager: "ArduinoJson"
   - Version: 6.21.3 or later

Installation Instructions:
1. Open Arduino IDE
2. Go to Tools > Manage Libraries...
3. Search for each library name
4. Click Install for each library
5. Restart Arduino IDE after installation

Note: Make sure to install all dependencies when prompted during library installation.
